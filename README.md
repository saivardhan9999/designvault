# DesignVault

DesignVault is a Python + Flask UI inspiration board and guided design direction builder inspired by Scalify's interactive AI website-ordering flow.

## Features

- Demo login and registration
- Per-user JSON vault storage
- Inspiration cards with categories, styles, scores, and palettes
- Guided brief builder for brand, industry, and style
- Generated design directions saved into the board
- Category and style filters
- Save, boost, and delete controls
- Dark neon SaaS UI with scan/build/order-style panels

## Tech Stack

- Python
- Flask
- HTML templates
- CSS
- Vanilla JavaScript reveal animation

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

Then open `http://127.0.0.1:5185`.

Authentication is demo-only and uses local JSON storage.
